# 국민생활도우미

국민에게 생긴 사건을 출발점으로 공공서비스, 기관, 준비정보, 절차, 반복 제출 데이터와 AI·제도개선 후보를 연결한 정적 웹 MVP입니다.

50개 사건 모두 정부24, 고용24, 보건복지부, 국민건강보험공단, 국가법령정보센터 등 공식 문서를 대조해 신청시기·핵심 절차·법적 근거·주의사항을 심화 보강했습니다.

## 현재 범위

- 50개 생활사건: 기존 20개 + 입양·보호아동·다문화·병역·교육·건강·산재·창업·노동권·주거·금융·생활안전 30개
- 사건·서비스·기관 통합검색
- 분야 필터
- 사건별 서비스·국민여정·준비정보·공식 출처
- 출생 사건 23개 서비스: 모든 출생가구 11개 + 돌봄·다자녀·저소득·장애·한부모·미숙아·고위험 임산부 등 조건부 12개
- 실직 사건 11개 서비스: 구직급여·재취업·건강보험·국민연금·긴급복지·체불임금 조건 분기
- 부모님 돌봄 사건 13개 서비스: 장기요양 판정 전후·재가·시설·치매·안전·의료비·통합돌봄·가족돌봄 분기
- 50개 전 사건 서비스별 적용조건·신청기한·책임기관·신청창구·국민 행동·처리결과·공식 원문
- 최대 3개 사건 비교
- 반복 제출정보·병목·AI 개선기회 진단
- 반응형 모바일 UI

> 비공식 프로토타입입니다. 50개 사건은 2026-07-12 기준 공식문서로 보강했지만, 개별 사건에 대한 행정기관의 판단이나 법률자문을 대신하지 않습니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 표시된 로컬 주소를 엽니다.

## 데이터 구조

원본 데이터는 `src/data/journeys.js`에 있습니다. 사건별 필수 항목:

- 기본정보: slug, title, category, summary, asOf
- services: 서비스, 기관, 이용자 행동
- steps: 국민 관점의 순차 여정
- documents: 준비정보·서류
- reuseCandidates: 정부 보유정보 재사용 후보
- friction: 서비스 단절·병목
- aiOpportunities: AI 및 제도개선 후보
- sources: 공식 출처

## 공개 전 필요한 것

1. 소관기관 또는 분야 전문가 내용 검수
2. 공식 기준 변경을 반영하는 최신성 점검 자동화
3. 공개 라이선스 결정

API 키, 서버, 데이터베이스는 현재 MVP 실행에 필요하지 않습니다.
